# 📧 AI Spam Email Detector

🔗 Live App: https://spam-email-detector-e2aeuujqssfmsufbnfmrzf.streamlit.app/

## 🚀 Overview
AI-powered Spam Email Detector that identifies spam and phishing emails using machine learning and rule-based analysis.

## ✨ Features
- 🤖 ML Model (TF-IDF + Logistic Regression)
- 🛡️ Rule-based phishing detection
- 📊 Risk scoring (Low / Medium / High)
- 💡 AI explanation of results
- 📥 Downloadable JSON report
- 🎨 Interactive Streamlit UI

## 🧠 How it works
1. Input email (sender, subject, body)
2. Text is cleaned and processed
3. ML model predicts spam/safe
4. Rule engine detects:
   - suspicious keywords
   - phishing phrases
   - fake domains
   - malicious links
5. Final decision + risk score generated

## 🛠️ Tech Stack
- Python
- Streamlit
- scikit-learn
- TF-IDF
- Logistic Regression
